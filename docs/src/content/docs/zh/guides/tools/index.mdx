LangCrew 提供了统一的工具管理系统，支持从不同来源发现、转换和使用工具。

## 选择您的工具模块

**[工具注册](/zh/guides/tools/tool-registry)** - 从这里开始基本工具使用  
自动发现和访问来自项目、CrewAI 和 LangChain 包的工具。

**[工具转换](/zh/guides/tools/tool-converter)** - 在工具格式间转换  
无缝转换 CrewAI 工具到 LangChain 格式，或将函数转换为工具。

**[MCP 集成](/zh/guides/tools/tool-mcp)** - 连接外部服务  
集成模型上下文协议服务器以扩展工具能力。

## 快速开始

刚接触 LangCrew 工具？从 **[工具注册指南](/zh/guides/tools/tool-registry)** 开始基本工具使用。

需要在工具格式间转换？查看 **[工具转换指南](/zh/guides/tools/tool-converter)**。

想连接外部服务？参考 **[MCP 集成指南](/zh/guides/tools/tool-mcp)**。
