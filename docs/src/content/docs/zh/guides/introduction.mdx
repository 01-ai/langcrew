## 什么是 LangCrew？

LangCrew 是一个**基于 LangGraph 构建的高级多智能体开发框架**，让创建、编排和监控 AI 智能体系统变得简单。它弥合了科研级多智能体系统与生产就绪应用之间的鸿沟。

## 为什么选择 LangCrew？

LangCrew 提供三大核心优势：

- **企业就绪**：内置人机协作审批流程和审计跟踪，满足生产环境需求
- **开发友好**：CrewAI 兼容装饰器，支持无缝迁移和快速开发
- **全栈可视化**：完整的 Agent-UI 协议，实时监控和 React 组件支持

### 开箱即用的功能

**装饰器开发模式** - 熟悉的 `@agent`、`@task`、`@crew` 语法  
**YAML 配置管理** - 外部化智能体和任务定义  
**工具审批工作流** - 敏感操作的人工监督  
**实时执行监控** - 实时执行流和调试功能  
**内存管理系统** - 智能体交互间的持久化上下文  
**LangGraph 集成** - 原生图形化工作流编排  
**生产安全保障** - 审计跟踪和审批机制  

## 核心功能

### 1. 熟悉的开发体验

使用您已经了解的装饰器语法：

```python
from langcrew import CrewBase, agent, task
from langcrew_tools import WebSearchTool, WebFetchTool

@CrewBase
class MarketingCrew:
    agents_config = "config/agents.yaml"
    tasks_config = "config/tasks.yaml"
    
    @agent
    def researcher(self):
        return Agent(
            config=self.agents_config["researcher"],
            tools=[WebSearchTool(), WebFetchTool()]
        )
    
    @task
    def market_analysis(self):
        return Task(
            config=self.tasks_config["market_analysis"],
            agent=self.researcher()
        )
```

### 2. 配置驱动开发

将业务逻辑与配置分离：

```yaml
# config/agents.yaml
researcher:
  role: "市场研究分析师"
  goal: "收集全面的市场情报"
  backstory: "专门识别市场趋势和机会的专家"

# config/tasks.yaml
market_analysis:
  description: "分析{product}的市场趋势"
  expected_output: "包含关键洞察的详细市场分析报告"
```

### 3. 人机协作控制

在需要的地方添加审批工作流：

```python
# 配置哪些工具需要审批
hitl_config = HITLConfig(
    enabled=True,
    interrupt_before_tools=["WebSearchTool", "DatabaseTool"]
)

crew = Crew(agents=[agent], tasks=[tasks], hitl=hitl_config)
```

### 4. 实时监控

将执行事件流式传输到前端：

```python
# 实时流式传输智能体执行过程
async for event in crew.astream_events(input_data):
    if event["event"] == "on_tool_start":
        print(f"工具启动: {event['name']}")
    elif event["event"] == "on_agent_finish":
        print(f"智能体完成: {event['data']['output']}")
```

## 从 CrewAI 迁移

LangCrew 提供现有 CrewAI 项目的**无缝迁移**：

1. **相同装饰器语法** - 您的 `@agent`、`@task`、`@crew` 装饰器无需修改
2. **配置兼容性** - 现有 YAML 配置可直接使用
3. **增强能力** - 自动获得 HITL、监控和生产级功能
