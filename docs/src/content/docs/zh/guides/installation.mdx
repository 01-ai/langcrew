本指南将指导您安装 langcrew 并设置开发环境。

## 系统要求

在安装 langcrew 之前，请确保您具备：

- **Python**: 3.11 到 3.12 版本
- **pip**: 建议使用最新版本
- **虚拟环境**: 建议用于项目隔离

## 安装

> **注意**: langcrew 目前不是开源的，需要访问私有包索引。

使用 uv 直接从私有索引安装 langcrew：

```bash
uv add langcrew --prerelease=allow --index https://nexus.lingyiwanwu.net/repository/pypi-hosted/simple
```

### 可选：永久配置私有索引

如果您计划频繁使用 langcrew，可以在 `pyproject.toml` 中配置私有索引，避免每次都指定：

```toml
[tool.uv]
default-groups = ['dev']
extra-index-url = ["https://nexus.lingyiwanwu.net/repository/pypi-hosted/simple"]
```

添加此配置后，您可以简单运行：

```bash
uv add langcrew --prerelease=allow
```

### 开发安装

要贡献或修改 langcrew，您需要克隆仓库并设置开发环境：

```bash
git clone https://github.com/01-ai/langcrew.git
cd langcrew
```

仓库在其 `pyproject.toml` 中包含必要的私有索引配置。使用 uv 安装开发依赖：

```bash
# 使用 uv（推荐）
uv sync

# 或使用 pip 与私有索引
pip install -e . --extra-index-url https://nexus.lingyiwanwu.net/repository/pypi-hosted/simple
```

## 验证安装

```python
import langcrew
print(langcrew.__version__)

# 测试基本功能
from langcrew import Agent

agent = Agent(
    role="测试智能体",
    goal="验证安装",
    backstory="一个有用的测试智能体"
)
print(f"智能体已创建: {agent.role}")
```
