## 快速示例

```python
from langcrew import Agent, Crew, Task
from langcrew.project import CrewBase, agent, crew, task
from langcrew_tools import WebSearchTool, WebFetchTool
from pydantic import BaseModel, Field
from typing import List

# 定义结构化输出模型
class MarketStrategy(BaseModel):
    name: str = Field(..., description="策略名称")
    tactics: List[str] = Field(..., description="营销策略")
    channels: List[str] = Field(..., description="分发渠道")

@CrewBase  # 使用装饰器定义智能体团队
class MarketingCrew:
    agents_config = "config/agents.yaml"  # 智能体配置文件
    tasks_config = "config/tasks.yaml"    # 任务配置文件
    
    @agent  # 装饰器自动注册智能体
    def market_analyst(self) -> Agent:
        return Agent(
            config=self.agents_config["market_analyst"],
            tools=[WebSearchTool(), WebFetchTool()],  # LangCrew 原生工具
            verbose=True
        )
    
    @agent  
    def strategist(self) -> Agent:
        return Agent(
            config=self.agents_config["strategist"],
            verbose=True
        )
    
    @task  # 装饰器自动注册任务
    def research_task(self) -> Task:
        return Task(
            config=self.tasks_config["research_task"],
            agent=self.market_analyst()
        )
    
    @task
    def strategy_task(self) -> Task:
        return Task(
            config=self.tasks_config["strategy_task"],  
            agent=self.strategist(),
            context=[self.research_task()],  # 任务依赖
            output_json=MarketStrategy      # 结构化输出
        )
    
    @crew  # 装饰器定义团队
    def crew(self) -> Crew:
        return Crew(
            agents=self.agents,  # 自动收集所有智能体
            tasks=self.tasks,    # 自动收集所有任务
            verbose=True,
            memory=True          # 启用记忆系统
        )

# 执行团队
inputs = {"domain": "AI自动化", "project": "企业营销"}
crew = MarketingCrew().crew()
result = crew.kickoff(inputs=inputs)
```

## 主要特性

- **多智能体协作** - 智能体协同工作，共享上下文和结果
- **灵活的执行器** - 从不同的执行策略中选择（顺序、并行、层次化）
- **人机协作** - 提供人工监督和干预的交互工具
- **生产就绪** - 为可扩展性而构建，支持异步和错误处理
- **可扩展** - 易于添加自定义智能体、工具和集成
