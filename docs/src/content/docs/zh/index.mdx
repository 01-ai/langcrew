| 方面 | LangGraph | CrewAI | LangCrew |
|------|-----------|---------|----------|
| **抽象层次** | 底层原语 | 高级模式 | **基于 LangGraph 的高级抽象** |
| **开发方式** | 从零构建 | 简单但有限 | **内置最佳实践** |
| **人机协作** | 基础中断/恢复 | 有限支持 | **高级审批系统 + 双语 UI** |

## 快速开始示例

**5分钟内构建你的第一个智能体团队：**

```python
from langcrew import Agent, Crew, Task
from langcrew.project import CrewBase, agent, crew, task
from langcrew_tools import WebSearchTool

@CrewBase
class ResearchCrew:
    @agent
    def researcher(self) -> Agent:
        return Agent(
            role="市场研究员",
            goal="发现AI自动化的最新趋势",
            tools=[WebSearchTool()],
            backstory="擅长发现市场洞察的专家"
        )
    
    @task
    def research_task(self) -> Task:
        return Task(
            description="研究 {topic} 的AI自动化趋势",
            agent=self.researcher(),
            expected_output="详细的市场分析报告"
        )
    
    @crew
    def crew(self) -> Crew:
        return Crew(
            agents=self.agents,
            tasks=self.tasks,
            memory=True,  # 启用记忆
            verbose=True  # 查看执行详情
        )

# 运行团队
result = ResearchCrew().crew().kickoff(
    inputs={"topic": "企业AI采用"}
)
```
