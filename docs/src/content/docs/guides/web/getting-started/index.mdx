Quick guide to get Web server working with your langcrew application.

## Installation

Web is included with langcrew - no additional installation required:

```bash
uv add langcrew --prerelease=allow --index https://nexus.lingyiwanwu.net/repository/pypi-hosted/simple
```

## Basic Example

Here's a simple example that creates a web server for an agent:

```python
from langcrew import Agent
from langcrew.web import create_server

# Create an agent
agent = Agent(
    role="Web Assistant",
    goal="Help users through web interface",
    backstory="You are a helpful web-based AI assistant",
    verbose=True
)

# Create and start server
server = create_server(
    agents=[agent],
    host="0.0.0.0",
    port=8000,
    cors_origins=["http://localhost:3000"]
)

# Start the server
if __name__ == "__main__":
    print("Starting web server on http://localhost:8000")
    server.run()
```

## LangGraph Integration Example

For more advanced workflows with LangGraph:

```python
from langcrew import Agent, Crew
from langcrew.web import create_langgraph_server

# Create agents and crew
researcher = Agent(
    role="Research Specialist",
    goal="Gather information on topics",
    backstory="Expert researcher with web access"
)

writer = Agent(
    role="Content Writer", 
    goal="Create engaging content",
    backstory="Skilled writer who creates compelling content"
)

crew = Crew(agents=[researcher, writer])

# Create LangGraph-enabled server
server = create_langgraph_server(
    crew=crew,
    host="0.0.0.0",
    port=8000,
    enable_streaming=True,
    tool_display=True
)

# Start server
server.run()
```

## Streaming Example

Enable real-time streaming responses:

```python
from langcrew.web import AdapterServer, LangGraphAdapter

# Create adapter with streaming
adapter = LangGraphAdapter(
    crew=crew,
    enable_streaming=True,
    stream_tool_calls=True,
    tool_display=True
)

# Create server
server = AdapterServer(
    adapter=adapter,
    host="0.0.0.0",
    port=8000
)

server.run()
```

## Testing Your Setup

### Test with cURL

```bash
# Test basic chat endpoint
curl -X POST http://localhost:8000/chat \
  -H "Content-Type: application/json" \
  -d '{"message": "Hello, can you help me with research?"}'

# Test task execution

  -H "Content-Type: application/json" \
  -d '{"task": "research_topic", "input": {"topic": "AI trends"}}'
```

### Test with SSE (Server-Sent Events)

```javascript
// Connect to streaming endpoint using SSE
const eventSource = new EventSource('http://localhost:8000/api/v1/chat', {
    method: 'POST',
    headers: {
        'Content-Type': 'application/json'
    },
    body: JSON.stringify({
        message: 'Hello from SSE!',
        session_id: 'test-session'

    const data = JSON.parse(event.data);
    console.log('Received:', data);
};

    console.error('SSE error:', error);
};
```

## Troubleshooting

### Server Won't Start

Check if port is available:
```bash
# Check if port 8000 is in use
lsof -i :8000

# Use different port if needed
server = create_server(port=8080)
```

### CORS Issues

Configure CORS origins properly:
```python
# ‚ùå Wrong - restrictive CORS
server = create_server(cors_origins=[])

# ‚úÖ Correct - allow your frontend
server = create_server(
    cors_origins=["http://localhost:3000", "https://yourdomain.com"]
)
```

### Streaming Not Working

Ensure streaming is enabled:
```python
# Enable streaming in adapter
adapter = LangGraphAdapter(
    crew=crew,
    enable_streaming=True,
    stream_tool_calls=True
)
```

### Import Errors

Make sure you're importing from the correct modules:
```python
# Core web components
from langcrew.web import create_server, create_langgraph_server

# Advanced components
from langcrew.web import AdapterServer, LangGraphAdapter

# Protocol types
from langcrew.web import ChatRequest, StreamMessage
```
